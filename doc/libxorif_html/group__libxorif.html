Returns
  • NULL if not found
  • Pointer to name string
Note
The short name is something like "oran_radio_if", and the returned result, is the fully qualified name, e.g. "oran_radio_if.a0000000". The present function defines additional masks/values which are used to split the RU ports into user/PRACH/SSB/... spatial streams.
