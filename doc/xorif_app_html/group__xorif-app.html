◆ command_set [1/2]

const struct command command_set[]

Command set defines the supported commands, functions and help text.

Note
Instructions... 1) Add commands to this data strucure 2) Format of each line is {name, function, text} 3) Multiple names can have same function (i.e. aliases) 3) Same name can have different functions (e.g. to split processing) 4) Functions can be NULL (used to provide additional help fields) 5) Text can be NULL (when no help is available, or for hidden commands) 6) If the first character of the text is '?' it is displayed as a 'usage' string. Otherwise the whole text is displayed as is.
